# WinReminder
## Hakkında
WinReminder, istenilen tarihler için kısa hatırlatıcı notlar tutmayı sağlayan Windows tabanlı basit ve işlevsel bir uygulamadır. Kullanıcı notunu yazar, tarihi seçer ve notunu metin belgesi olarak kolayca kaydeder.
## Kullanılan teknolojiler
- Visual C++ 2026 v14.51
- Windows 11 SDK v10.0.28000
- Microsoft Foundation Classes
## Derleme ve çalıştırma
- Repoyu klonlayın.
- "C++ masaüstü geliştirme" iş yükü kurulu olan Visual Studio 2026'da **wr64.slnx** dosyasını açın.
- Halihazırda Release konfigürasyonunda olan projeyi derleyerek kolayca çalıştırabilirsiniz.

**Not:** Yakın zamanda x64 mimarisi için MSI kurulum paketi yayınlamayı planlıyorum.
